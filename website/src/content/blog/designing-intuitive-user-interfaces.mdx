# Good interfaces reduce decisions

Users should not need a manual. Every action should make sense from context and feedback.

## Heuristics that hold up

- Use one visual priority per section
- Keep action labels specific and short
- Make system state visible after every mutation

A clean interface is not minimalism for its own sake. It is deliberate reduction of cognitive load.
